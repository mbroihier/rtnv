Pressing the "Enter" Button

This test category contains test cases intended to verify the requirements associated with pressing the enter button.
  1. ENTER-010 - Pressing Enter when there is a Valid Ticker Symbol and Valid Number of Shares
  2. ENTER-020 - Pressing Enter when there is an Invalid Ticker Symbol
  3. ENTER-030 - Pressing Enter when there is no Ticker Symbol
  4. ENTER-040 - Pressing Enter when there is a Valid Ticker Symbol but no Number of Shares Entered
  5. ENTER-050 - Pressing Enter when there is a Valid Ticker Symbol that matches a previously entered Ticker Symbol
