Test Case NEXT-010
System Test PHS
Title First button press displays first portfolio entry
Objective This test case is intended to verify the following implied or specified requirements:
  • When the next button is pressed and it is the first press after any other button press, rtnv shall display the first stock in the the persistent copy of the portfolio.
This test case verifies that the app displays the first entry of the portfolio if it is the first "Next" press in a sequence of other actions.
Setup
Actions
  1. Delete the app from the target device.
  2. Compile and load rtnv onto the target Android device.
  3. Verify the compile completes with no errors.
  4. Verify the installation completes with no errors.
  5. Select the "Ticker Symbol" field and enter the ticker value of AAPL.
  6. Select the "Number of Shares" field and enter the value 200.
  7. Press the "Enter" button.
  8. Verify a "blank" sceen is displayed.
  9. Select the "Ticker Symbol" field and enter the ticker value of INTC.
  10. Select the "Number of Shares" field and enter the value 500.
  11. Press the "Enter" button.
  12. Verify a "blank" screen is displayed.
  13. Press the "Next" button and verify that the symbol AAPL is displayed with 200 shares and that a non zero value is displayed for the last closing price of the stock.
  14. Select the "Ticker Symbol" field and overwrite the AAPL entry with SBUX.
  15. Select the "Number of Shares" field and overwrite the field entry with 800.
  16. Press the "Next" button and verify that the symbol AAPL is displayed with 200 shares and that a non zero value is displayed for the last closing price of the stock.
  17. Select the "Ticker Symbol" field and overwrite the AAPL entry with NFG.
  18. Select the "Number of Shares" field and overwrite the field entry with 300.
  19. Press the "Next" button and verify that the symbol AAPL is displayed with 200 shares and that a non zero value is displayed for the last closing price of the stock.
Expected When the first "Next" of a button push sequence is entered, the first entry of the portfolio is entered.
Results PASSED
Cleanup Unnecessary
Requirements RTNV-PR-NEXT-010
