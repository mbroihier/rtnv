Test Case ENTER-040
System Test PHS
Title Pressing Enter when there is a Valid Ticker Symbol but no Number of Shares Entered
Objective This test case is intended to verify the following implied or specified requirements:
  • When the enter button is pressed, rtnv shall send a retrieve request to the public server if the ticker field is not empty.
  • rtnv shall store a persistent copy of the ticker, number of shares, and value of each share in the persistent database.
  • rtnv shall store zero as the number of shares if the number of shares was not entered by the user.
  • rtnv shall clear the ticker, number of shares, and value of each share fields at completion of processing.
The objective of this test case is to verify the requirements identified above which cover the entry of a stock into the portfolio which exists in the public server, but the user did not specify the number of shares in the portfolio.
Setup
  • Test case ENTER-030 was immediately executed prior to this test case.
Actions
  1. Select the "Ticker Symbol" field and enter the value JNJ.
  2. Press the "Enter" button.
  3. Verify a "blank" screen is displayed.
  4. Press the "Next" button and verify that the symbol AAPL is displayed with 200 shares and that a non zero value is displayed for the last closing price of the stock.
  5. Press the "Next" button again and verify that the symbol INTC is displayed with 500 shares and that a non zero value is displayed for the last closing price of the stock.
  6. Press the "Next" button again and verify that the symbol AAA is displayed with 1 share and that "Not found" is displayed for the last closing price of the stock.
  7. Press the "Next" button again and verify that the symbol JNJ is displayed with 0 shares and a non zero value is displayed for the last closing price of the stock.
  8. Press the "Next" button again and verify that a "blank" screen is displayed.
Expected If the number of shares is not defined by the operator, the default number of shares is zero.
Results PASSED
Cleanup Unnecessary
Requirements RTNV-PR-ENTER-010, RTNV-PR-ENTER-010.001, RTNV-PR-ENTER-010.003, RTNV-PR-ENTER-010.005
