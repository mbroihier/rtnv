Test Case ENTER-050
System Test PHS
Title Pressing Enter when there is a Valid Ticker Symbol that matches a previously entered Ticker Symbol
Objective This test case is intended to verify the following implied or specified requirements:
  • When the enter button is pressed, rtnv shall send a retrieve request to the public server if the ticker field is not empty.
  • rtnv shall store a persistent copy of the ticker, number of shares, and value of each share in the persistent database.
  • The ticker shall be the primary and unique key of the persistent store (previous definitions will be replaced).
  • rtnv shall clear the ticker, number of shares, and value of each share fields at completion of processing.
The objective of this test case is to verify the requirements identified above which cover the entry of a stock into the portfolio which exists in the portfolio. The old entry should be removed and replaced by the new entry.
Setup
  • Test case ENTER-040 was immediately executed prior to this test case.
Actions
  1. Select the "Ticker Symbol" field and enter the value JNJ.
  2. Select the "Number of Shares" field and enter the value 100.
  3. Press the "Enter" button.
  4. Verify a "blank" screen is displayed.
  5. Press the "Next" button and verify that the symbol AAPL is displayed with 200 shares and that a non zero value is displayed for the last closing price of the stock.
  6. Press the "Next" button again and verify that the symbol INTC is displayed with 500 shares and that a non zero value is displayed for the last closing price of the stock.
  7. Press the "Next" button again and verify that the symbol AAA is displayed with 1 share and that "Not found" is displayed for the last closing price of the stock.
  8. Press the "Next" button again and verify that the symbol JNJ is displayed with 100 shares and a non zero value is displayed for the last closing price of the stock.
  9. Press the "Next" button again and verify that a "blank" screen is displayed.
Expected If a duplicate ticker is entered, the new definition overides the old definition.
Results PASSED
Cleanup Unnecessary
Requirements RTNV-PR-ENTER-010, RTNV-PR-ENTER-010.001, RTNV-PR-ENTER-010.002, RTNV-PR-ENTER-010.005
